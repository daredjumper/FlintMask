# flint.mask

> 🔥 Simple URL masking — part of the [flint.dev](https://github.com/daredjumper/FlintV1) ecosystem.

> [!WARNING]
> **Work in Progress** — features may be incomplete, broken, or subject to change.

---

## What is flint.mask?

**flint.mask** is a URL masking utility that hides some of the traffic of the destination of a link through embed-based redirection. It's designed to keep the address bar clean, obscure referrer data, and make links look like something they're not.

---

## Features

- **Clean UI** — dark glass-morphism design consistent with the rest of Flint; nothing looks out of place
- **Previous link memory** — remembers recently masked URLs locally so you don't have to re-paste them
- **about:blank opener** — loads masked content through an `about:blank` context to strip referrer and obscure origin
- **Embed-based masking** — wraps destinations in iframe/embed layers to prevent direct URL exposure

## Disclaimer

flint.mask is intended for legitimate use cases such as link privacy, referrer stripping, and URL shortening/masking. Use responsibly.
